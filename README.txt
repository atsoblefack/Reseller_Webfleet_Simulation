
Instructions pour déployer l'application avancée Webfleet Cameroun :

1. Installer les dépendances :
   pip install -r requirements.txt

2. Lancer l'application :
   streamlit run app.py

3. Déploiement en ligne :
   - Créer un compte sur https://streamlit.io/cloud
   - Uploader les fichiers : app.py, requirements.txt, App_Webfleet_Projection_FCFA.xlsx
   - Lancer le déploiement
